Legacy CLI

The Siteleaf v1 CLI and Ruby Gem is not compatible with v2 sites (and vice versa), so we recommend using a Gemfile to allow both versions to be installed.

For documentation on the v1 CLI, see:
https://github.com/siteleaf/siteleaf-gem/tree/v1 →

For information on the v2 CLI, see:
http://learn.siteleaf.com/themes/cli/ →
